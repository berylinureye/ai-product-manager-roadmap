# AI Product Manager Learning Roadmap

这个仓库用于记录我转向 AI 产品经理过程中的系统学习、章节笔记和核心概念笔记。现阶段先专注学习，不把作品集、GitHub 排行榜分析或独立项目混在这个 roadmap 里。

## 当前学习目标

- 建立 AI 产品经理需要的基础语言系统：LLM、Prompt、Agent、RAG、MCP、评测集、bad case、fallback、成本与效果取舍。
- 用 Datawhale、B 站课程和低代码工具补齐 Agent/RAG 的原理与产品判断。
- 通过章节笔记沉淀自己的理解、错误纠偏和可复用表达。

## 仓库结构

- `datawhale-hello-agents/`：Datawhale Hello-Agents 逐章学习笔记。
- `8-week-learning-plan.md`：8 周学习路线，只保留学习主线。
- `course-based-plan.md`：按课程和开源教程组织的学习材料顺序。
- `assets/`：学习过程中用到的图片或辅助材料。

## 当前重点材料

- Datawhale Hello-Agents：https://github.com/datawhalechina/hello-agents
- Datawhale All-in-RAG：https://github.com/datawhalechina/all-in-rag
- AI 产品经理认知、Prompt、Coze、Dify、RAGFlow、MCP 等课程材料。

## 记录规则

- 系统教程按资料来源单独建文件夹，例如 `datawhale-hello-agents/`。
- 每次学习结束后更新对应章节笔记，并在笔记里记录日期、理解、纠偏和下次继续位置。
- 独立项目、GitHub 排行榜分析、作品集材料暂时不放在这个 roadmap 里。
- 每篇笔记尽量记录：学了什么、我怎么理解、哪里答错或混淆、下一步要补什么。
