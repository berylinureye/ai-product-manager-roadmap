# AI Product Manager Roadmap

这个文件夹用于记录陈卓欣转向 AI 产品经理的 8 周学习、作品集和每周复盘。

## 当前定位

目标岗位：AI 产品经理，优先切入 `AI 应用型产品 / Agent 工作流 / 金融风控或教育场景 AI 产品`。

现有优势：

- 金融策略产品经历：美团月付贷中策略、大促风控、提额、分层、指标监控。
- 数据分析底子：数学统计 + 金融背景，SQL/Python/R，Kaggle Home Credit 铜牌。
- AI 动手经历：A-Level Plus AI 拍照批改产品，涉及 OCR、多模型协作、Agent 投票、Python 规则校验、埋点与回归测试。
- 英语与海外背景：适合补充出海 AI 产品、海外竞品研究、英文资料阅读。

主要短板：

- GitHub 公开作品少，目前主页只有 1 个公开仓库，外界难以快速判断你的持续学习和动手能力。
- AI 项目需要从“我做过”升级为“能展示、能复盘、能讲清原理和指标”。
- 需要系统补齐 AI PM 高频语言：模型边界、Prompt、Agent、RAG、MCP、评测集、bad case、fallback、成本/效果取舍。
- 简历需要围绕目标岗位重写：突出 AI 应用落地、数据驱动、策略产品、端到端 demo。

## 8 周最终交付物

1. 一份新版 AI 产品经理简历。
2. 8 篇每周学习记录，可同步到 GitHub。
3. 8 篇 GitHub 周榜产品观察，记录每周 3-5 个 AI/Agent/PM 相关项目。
4. 1 个可演示 Agent/RAG demo，包含产品说明、流程图、prompt、工具列表、失败兜底。
5. 1 份完整 AI 功能 PRD，建议主题：`AI 提升跨境下单转化` 或 `A-Level Plus 拍照批改优化`。
6. 1 份面试作品集文档：项目背景、目标用户、方案选型、指标、评测集、bad case、迭代计划。

## 每天 3 小时结构

- 45 分钟：学习一个 AI PM 知识点。
- 60 分钟：动手搭建、体验产品、跑数据或写 PRD。
- 45 分钟：输出成文档、图表、GitHub commit。
- 30 分钟：复盘，记录今天学会了什么、还不会什么、明天怎么补。

## 核心学习材料

- B 站课程：先用于建立概念和工具手感，不完整刷课。
- Datawhale Hello-Agents：https://github.com/datawhalechina/hello-agents
- Datawhale All-in-RAG：https://github.com/datawhalechina/all-in-rag

材料使用顺序：

1. 先看 AI PM / Prompt 课程，知道岗位要什么。
2. 再用 Coze / Dify / RAGFlow 跑通低代码 demo。
3. 然后用 Hello-Agents 和 All-in-RAG 补 Agent/RAG 原理、评估和失败模式。
4. 最后转成 A-Level Plus、跨境转化 PRD、GitHub 周榜 Agent 三个作品。
