# 7 天产品思维训练

这个文件夹用于记录 7 天产品思维训练的交互过程、案例回答、纠偏和最终表达。

训练目标：

- 补足面试中暴露的产品思维短板：需求判断、优先级、竞品分析、产品 sense。
- 把传统产品方法论和 AI 时代的 PM skills 结合起来。
- 形成可复习、可面试复用、可持续上传 GitHub 的学习笔记。

## 训练方式

每一天固定流程：

1. 讲一个小概念。
2. 给一个产品 case。
3. 你像面试一样回答。
4. 我判断产品思维薄弱点。
5. 我纠偏并给出更好的表达。
6. 把“原始回答 + 纠偏 + 最终表达”写入当天笔记。
